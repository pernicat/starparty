# starparty

Utility for planning optimal dates for Star Parties and astrophotography

## Overview

StarParty is a Jupyter notebook-based tool that helps astronomers and stargazers identify the best dates and times for observing the night sky during the summer months. By analyzing lunar phases and providing curated lists of celestial objects, StarParty makes it easy to plan memorable star parties.

## Features

- **Moon Phase Analysis**: Calculates lunar phases for each day and time during the summer season
- **Optimal Date Prediction**: Identifies the best nights for observation based on moon illumination and phase
- **Celestial Object Recommendations**: Provides curated lists of deep-sky objects, stars, and other targets ideal for viewing on each good observation date
- **Interactive Notebook**: Easy-to-use Jupyter notebook interface for exploring and customizing your observing plans

## Getting Started

1. Clone the repository
2. Open the Jupyter notebook
3. Specify your target dates and location
4. Review moon phase data and recommended observation targets
5. Plan your star party!
